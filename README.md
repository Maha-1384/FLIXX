# FLIXX Movie App

A dynamic movie application that allows users to search for movies, view details, and read reviews.

## Features
- Search for movies using a third-party API.
- View movie details such as ratings, cast, and synopsis.
- Interactive and user-friendly UI.

## Technologies Used
- HTML, CSS, JavaScript
- API Integration
- Responsive Web Design

## Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Open `index.html` in your browser.

## Usage
- Enter a movie name in the search bar to view details.
